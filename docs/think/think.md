# 不同阶段的想法

## 2022-03-15 不喜欢文革背景

不喜欢文革背景的小说，也不喜欢把人性写得太表面的小说，哗众取宠，读腻了无趣，时间太短了，交给新鲜的书吧。

## 2022-04-08 小说阅读顺序

1. 先简要了解下人物关系及小说整个思想和故事概要
2. 通读原著
3. 观看对应的影视作品加强理解
4. 看其他人解析这本书
5. 再读原著，分析自己想要着重了解的内容

## 2022-05-23 国外小说的阅读方式

1. 故事梗概
2. 主要人物的关系图
3. 所有人物的关系图
4. 读 10 章左右再重温这 10 章
5. 通读原著

人名真的真的太难记了，经常读着读着就忘记了到底在讲什么故事，一定要在网上找到关系图，然后对照着读。
