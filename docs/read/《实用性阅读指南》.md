# 《实用性阅读指南》


## 第一章 有效率的读书

```
1.1 确定读书目的
  为什么想要读它
1.2只摘取自己需要的信息
  清楚目的
  通过概览目录后记了解全书内容
  确定问题
1.3 通过序言目录和后记掌握概要
1.4 要有时间意识
  会让自己锻炼短时间内提炼信息 提升效率 摘取信息
1.5 不试图全部理解
  重要信息 只有 20%，这 20%中重要的只有 4%。
1.6 有意识地输出
  向他人讲述和写下来。
1.7 争取多读
  每次打开时会思考读到哪了，讲了什么
1.8 不被他人意见左右
1.9 不借书
  买书
```

```
2.7 如何制作一本书的思维导图

```

## 第四章 将书中内容付诸

```
4.1 知识，只有付诸行动才可以发挥效果
  例如，读了关于阅读的书，改变读书方法。读了关于宣传语的书，修改博客标题
4.2 将自我指导用于读书
4.3 从书本上设定行动目标
  用四格法和 smart 法则
4.4 想象自己达到目标的样子
  状态 心情 看见什么 感觉到什么
4.5 明确现在的位置
  现在是目标的百分多少
4.6 思考与行动目标之间的差距
  比如，50%的时候，是一种怎样的状态？
4.7 为了填补差距的具体行动
  为了达到加 10%的效果，我能做些什么，列举 5 个
4.8 挑出一项最想做的
4.9 将行动细分成从明天就能开始的
4.10 确定需要实施的事情
4.11 将行动告知他人
4.12 付诸行动
```
