# 前言一、为什么要读这本书

欧丽娟的红楼梦公开课最开始我在 2020-04-20 就已经听过并记笔记了，没想到原来早在 2018 年 8 月就已经出了实体书《大观红楼》，到底还是我没用心没有细致的查资料，没能有机会好好的研究研究。对比了讲义时间是 2012 年，所以录制公开课的时间猜测应该是 2012 年的样子，正好过了 10 年，也算是缘分吧。
